# [Initialisation du Projet]

## 1. Contexte & Discussion (Narratif)
Nous démarrons le projet "Semantic Search MCP". L'objectif est de fournir un outil de recherche sémantique local pour les agents. Cette tâche couvre la mise en place initiale de la structure du repository, la configuration de Python (pyproject.toml) et la création de la documentation de base.

## 2. Fichiers Concernés
- `pyproject.toml`
- `README.md`
- `src/__init__.py`
- `.gitignore`

## 3. Objectifs (Definition of Done)
- Le repository est initialisé (`git init`).
- La structure de dossiers est en place.
- Les dépendances initiales sont définies (`mcp`, `fastembed` ou équivalent).
- Le linter et les outils de développement sont configurés.
